Overview

Encrypted networking for everyone

Open

Anyone can join by peering with someone already connected.

P2P

Cjdns is built on the same principles as Kademlia DHT, the network is censorship resistant.

Secure

Packets are encrypted end-to-end using a clever public key based on your unique cjdns ipv6 address.

How do I join?

Install the cjdns software on a supported device/platform and locate a peer.

  • Ubuntu
  • Archlinux
  • Fedora
  • OpenWRT
  • Android
